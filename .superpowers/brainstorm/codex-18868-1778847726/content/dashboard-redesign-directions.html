Arah redesign dashboard GitHire

Pilih arah visual dan struktur kerja untuk semua menu user dashboard. Opsi A adalah rekomendasi saya untuk GitHire karena fokusnya career workflow, bukan sekadar tampilan cantik.

A. Career Command Center

Dashboard terasa seperti pusat kontrol pencarian kerja: status profil, rekomendasi job, skill gap, roadmap, dan lamaran disusun berdasarkan langkah berikutnya.

B. Premium Bento Visual

Lebih ekspresif dan modern: card besar, visual match score, warna kuat, dan hierarchy asimetris. Bagus untuk demo hackathon, tapi perlu dijaga agar tidak ramai.

C. Clean SaaS Productivity

Lebih terang, padat, dan operasional: cocok kalau targetnya dashboard harian yang cepat discan, table-heavy, dan terasa seperti product SaaS profesional.
